# 🏠 Imobiliária MVC

Projeto desenvolvido em **ASP.NET Core MVC** para a disciplina de Programação I.

O sistema simula uma aplicação de gerenciamento de imóveis, permitindo o cadastro, edição, visualização e busca de imóveis, com upload de imagens e categorização.

## 🎯 Objetivo do projeto

Desenvolver uma aplicação web utilizando o padrão **MVC (Model-View-Controller)**, aplicando conceitos de:

- Estrutura de aplicações web
- CRUD completo
- Organização em camadas (Model, View e Controller)
- Manipulação de listas em memória
- Upload de imagens
- Estilização de interface com CSS

---

## 🧱 Tecnologias utilizadas

- ASP.NET Core MVC
- C#
- HTML5
- CSS3
- Bootstrap
- Razor Views

---

## ⚙️ Funcionalidades

- Cadastro de imóveis
- Edição de imóveis
- Exclusão de imóveis
- Visualização de detalhes
- Listagem de imóveis
- Busca por múltiplos campos (título, descrição, categoria, endereço, etc.)
- Upload de imagens dos imóveis
- Cadastro de cômodos por imóvel

---

## 🎨 Interface

O projeto possui um layout personalizado com:

- Tema escuro com degradê cinza
- Destaques em dourado
- Inputs estilizados com transparência
- Botões padronizados
- Tabelas com melhor legibilidade
- Cards para exibição de detalhes

---

## 📁 Estrutura do projeto

- Controllers → Regras de controle da aplicação  
- Models → Entidades do sistema (Imóvel, Cômodo, etc.)  
- Views → Interface do usuário  
- wwwroot → Arquivos estáticos (CSS e imagens)
